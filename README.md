# Programmer's Hub – Q&A Web Application
  Programmer's Hub is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) that allows users to post technical questions and answer others in a secure, interactive platform.

>## Features
- [X] User registration and login with **JWT** authentication
- [X] Create, view, and manage **questions** and **answers**
- [X] Search questions by **title**
- [X] Secure password hashing using **bcrypt**
- [X] Protected routes and session persistence
- [X] Clean and simple UI for easy navigation
- [X] Clear session and redirect to login page.

>## Technologies Used
- **Frontend :** React.js, Axios
- **Backend :** Node.js, Express.js
- **Database :** MongoDB
- **Authentication :** JWT, bcrypt     


>## How to Run Locally

> #### 1. Clone the repository :
```ruby
git clone https://github.com/your-username/programmers-hub.git
cd programmers-hub
```
> #### 2. Install dependencies :
```ruby
cd "front end"
npm install
cd ..
cd back-end
npm install
```
> #### 3. Create a `.env` file in the back-end folder and add :
```ruby
PORT = your-port
MONGO_URI=your-mongodb-connection-string
JWT_SECRET=your-jwt-secret-key
```
> #### 4. Run the server :
```ruby
cd back-end
npm start
```
>- [ ] Open new terminal
```ruby
cd "front end"
npm start
```

> #### 5. Open your browser and visit: ```http://localhost:your-port```
